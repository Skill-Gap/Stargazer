Integrate Home Assistant into Stargazer.

Environment Variables:

HA_URL=http://192.168.1.35:8123
HA_TOKEN=<read from process.env>
PI_PLUG_ENTITY=switch.pi_socket_1
DESKTOP_PLUG_ENTITY=switch.desktop_socket_1
LAMP_ENTITY=switch.ihome_ww117_smart_plug_socket_1

Backend Requirements:

Create routes:

GET /api/homeassistant/status

GET /api/homeassistant/pi

GET /api/homeassistant/desktop

GET /api/homeassistant/lamp

POST /api/homeassistant/pi/on
POST /api/homeassistant/pi/off

POST /api/homeassistant/desktop/on
POST /api/homeassistant/desktop/off

POST /api/homeassistant/lamp/on
POST /api/homeassistant/lamp/off

Use Home Assistant REST API.

Status endpoint should verify Home Assistant is reachable.

Device endpoints should return:
- entity_id
- state
- friendly_name

Frontend Requirements:

Add Home Assistant to Stargazer sidebar.

Create cards:

Home Assistant Status
Pi Plug
Desktop Plug
Lamp

Each card should show:
- Current state
- Last updated

Each device card should have:
[ON] [OFF]

Use the same visual style as the Systems dashboard.

Do not hardcode tokens.
Read all secrets from environment variables.
